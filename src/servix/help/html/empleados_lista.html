Panel de Gestión de Empleados

Desde este panel, el encargado puede administrar el personal del restaurante. La tabla muestra información detallada de cada empleado registrado en el sistema.

Información de la Tabla

La tabla contiene la siguiente informacion:

  • ID: Identificador único del sistema.
  • Datos Personales: Nombre, primer apellido y segundo apellido.
  • Contacto: Número de teléfono y correo electrónico.
  • Usuario: Nombre de usuario utilizado para acceder a la aplicación.

Acciones Disponibles

En la parte inferior dispone de los botones para gestionar los registros:

  • Añadir: Abre un formulario para dar de alta a un nuevo trabajador.
  • Editar: Permite modificar los datos del empleado seleccionado en un formulario.
  • Eliminar: Borra al empleado seleccionado tras una confirmación. Nota: Esta acción es irreversible.
  • Volver: Regresa a la interfaz principal del encargado.
Importante: Para editar o eliminar, debe hacer clic primero sobre la fila del empleado correspondiente en la tabla. Si no selecciona ninguna, el sistema mostrará un aviso.
La tabla se actualiza automáticamente después de realizar cualquier cambio (alta, baja o edición).
