Asignaciones restaurante-encargado

La herramienta de asignaciones permite vincular operativamente a los Encargados con los Restaurantes. Esta unión es fundamental para determinar qué locales puede gestionar cada empleado al iniciar sesión.

Proceso de Vinculación

Para crear una nueva relación laboral, el sistema utiliza un método de selección simplificado:

  1. Selección de Encargado: Elija al usuario desde el menú desplegable. El sistema muestra el nombre completo del personal disponible.
  2. Selección de Restaurante: Elija el establecimiento de destino. El listado muestra el nombre del local junto a su dirección para evitar confusiones.
  3. Confirmación: Pulse el botón Asignar.

Control

En la tabla podrá supervisar todas las asignaciones vigentes en tiempo real.

Eliminar Asignaciones

Si un encargado deja de prestar servicio en un local, puede romper el vínculo seleccionando la fila correspondiente y pulsando Eliminar Asignación.

  • Esta acción no elimina al usuario ni al restaurante, solo la relación entre ambos.
Actualización Dinámica: Tras cada asignación o eliminación, el sistema refresca los desplegables para evitar errores de selección.
