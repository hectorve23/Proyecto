Gestión de Encargados

El sistema permite al Gerente crear y administrar cuentas de usuario de tipo Encargado. Estas cuentas son esenciales para que el personal pueda acceder a sus funciones de gestión en sus respectivos locales.

Proceso de Alta y Validación

Para garantizar la integridad de la base de datos, el sistema aplica las siguientes reglas automáticas antes de permitir el registro:

  • Campos Obligatorios: No se permiten campos vacíos (Nombre, Apellidos, Usuario, etc.).
  • Validación de Teléfono: Debe contener exactamente 9 dígitos numéricos.
  • Validación de Email: Debe seguir un formato estándar (ejemplo@dominio.com).
  • Cotejo de Contraseña: Los campos de "Contraseña" y "Confirmación" deben coincidir exactamente.

Seguridad de Acceso

Servix prioriza la seguridad de los datos de acceso:

Encriptación

Al pulsar el botón Validar Encargado, el sistema utiliza un algoritmo de Hash para encriptar la contraseña antes de almacenarla en la base de datos. Esto impide que las contraseñas sean visibles en texto plano, incluso para el administrador de la base de datos.

Gestión de Perfiles

Para modificar un encargado existente, selecciónelo en la tabla y pulse Editar. Esto abrirá un diálogo que permite actualizar sus datos personales o de contacto.

Atención: Si elimina a un encargado mediante el botón Eliminar, se perderá su acceso al sistema de forma inmediata. Asegúrese de que el usuario seleccionado es el correcto antes de confirmar la operación en el cuadro de diálogo.
