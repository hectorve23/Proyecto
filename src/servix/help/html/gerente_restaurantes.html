Gestión de Restaurantes

Esta sección permite el control administrativo sobre la infraestructura física de Servix. Aquí podrá registrar nuevos locales o modificar la configuración operativa de los existentes.

Campos de Información

Cada restaurante se define por los siguientes parámetros obligatorios:

Campo Descripción / Formato
ID Identificador único numérico (autogenerado).
Capacidad Número máximo de comensales permitidos (valor entero).
Horarios Rango de apertura y cierre gestionado mediante selectores numéricos.
Contacto Teléfono (9 dígitos) y dirección de correo electrónico válida.

Operaciones Disponibles

1. Alta de Restaurante

Para añadir un local, debe completar todos los campos de texto y ajustar los valores de capacidad y horario. El sistema lo insertará en la base de datos y actualizará la tabla automáticamente.

2. Edición de Datos

Para modificar un restaurante:

  1. Seleccione una fila en la tabla de restaurantes.
  2. Pulse el botón Editar.
  3. Se abrirá una ventana emergente con los datos cargados para su modificación.

3. Eliminación Directa

Al pulsar Eliminar, el sistema identifica el ID seleccionado. Esta acción utiliza una transacción segura con opción de Rollback en caso de fallo crítico en la base de datos.

Refresco Automático: Tras cualquier operación exitosa, el sistema se actualiza para asegurar que los datos mostrados son los más recientes.
Aviso de Seguridad: La eliminación de un restaurante es permanente y afectará a todas las asignaciones de encargados vinculadas a dicho local.
