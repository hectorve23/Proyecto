Nueva Mesa

Utilice este formulario para registrar una nueva unidad de servicio en el restaurante.

Instrucciones de Llenado

  • Capacidad: Ingrese el número total de personas que pueden sentarse en la mesa simultáneamente.
Regla de Validación: El sistema no permitirá registrar mesas con capacidad 0 o campos vacíos. Debe ser un número entero positivo.

Resultado de la Acción

Al pulsar Añadir:

  1. La mesa se guardará en la base de datos.
  2. El estado inicial será automáticamente "Disponible".
  3. La ventana se cerrará y la tabla principal se actualizará automáticamente.

Si pulsa Cancelar, se cerrará el formulario sin guardar ningún cambio.
