Edición de Mesas

Este panel permite modificar las capacidad de las mesas ya registradas en el sistema.

Cómo editar una mesa

  1. Selección del ID: El desplegable muestra el ID de la mesa seleccionada inicialmente. Puede cambiarlo para editar una mesa distinta sin salir de la ventana.
  2. Carga de datos: Al cambiar el ID, el campo "Capacidad" se actualizará automáticamente con el valor actual almacenado en la base de datos.
  3. Modificación: Escriba el nuevo número de comensales en el cuadro de texto.
Información: Cambiar la capacidad de una mesa no afecta a su estado actual (Libre/Ocupada), solo a su ficha técnica.

Confirmación

  • Pulse Aceptar para guardar los cambios permanentemente.
  • Pulse Cancelar para descartar las ediciones y volver al listado general.
