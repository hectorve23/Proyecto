Gestión de Mesas del Establecimiento

Desde este panel, el encargado puede controlar la disposición y disponibilidad de las mesas en el sistema.

Acciones Disponibles

  • Añadir mesa: Abre un formulario para dar de alta una nueva mesa indicando su capacidad.
  • Editar mesa: Permite modificar la capacidad de la mesa seleccionada en la tabla.
  • Eliminar mesa: Borra permanentemente la mesa seleccionada. Atención: Esta acción no se puede deshacer.
Importante: Para editar o eliminar, primero debe hacer clic sobre la fila correspondiente en la tabla de datos.

Interpretación de la Tabla

Columna Descripción
ID Mesa Identificador único numérico de la mesa.
Capacidad Número máximo de comensales permitido.
Estado Indica si la mesa está libre (true) u ocupada/reservada (false).
La tabla se recarga automáticamente tras realizar cualquier operación para mostrar siempre la información real de la base de datos.
