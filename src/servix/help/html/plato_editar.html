Modificación de Platos

Esta ventana permite actualizar la información de un plato seleccionado previamente en el menú. Los cambios realizados se verán reflejados inmediatamente en la base de datos del sistema.

Campos Disponibles

Puede modificar cualquiera de los siguientes parámetros:

  • Nombre: Permite cambiar el nombre del plato. Tenga en cuenta que si lo cambia, este será el nuevo identificador.
  • Precio: Utilice el control numérico para ajustar el coste del plato.
  • Categoría: Desplegable para reubicar el plato en una sección distinta (Bebidas, Entrantes, etc.).

Acciones del Formulario

Para finalizar la edición, dispone de dos opciones:

  • Validar: Aplica los cambios y actualiza el registro en la base de datos. Si el proceso tiene éxito, se mostrará un mensaje de confirmación y volverá a la gestión del menú.
  • Cancelar: Cierra la ventana actual sin guardar ninguna modificación, regresando a la lista de platos.
Validación de datos: Al igual que en la creación, el sistema requiere que el nombre no esté vacío y que el precio sea estrictamente mayor a 0.
Si el plato está siendo editado por error, puede pulsar Cancelar o cerrar la ventana para mantener los datos originales intactos.
