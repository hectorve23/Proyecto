Gestión Integral de Reservas

Este panel permite organizar las peticiones de mesa de los clientes mediante un sistema de pestañas por estado.

1. Pestaña de Reservas Pendientes

Aquí aparecen las reservas que aún no tienen una mesa asignada.

  • Proceso de Confirmación: Seleccione una reserva de la lista y pulse el botón Confirmar.
  • Validación automática: El sistema verificará si hay una mesa disponible con capacidad suficiente para la fecha y hora indicadas.
  • Aviso: Si el sistema muestra el mensaje "No hay mesa disponible", deberá esperar a que se libere espacio o gestionar la mesa manualmente.

2. Pestaña de Reservas Confirmadas

Muestra las reservas que ya tienen una mesa vinculada con éxito.

  • Pulse Informe para generar un listado detallado de todas las reservas aceptadas en el sistema.

3. Pestaña de Reservas Rechazadas

Histórico de reservas que fueron canceladas por el cliente o rechazadas por falta de aforo.

  • También permite generar un informe específico para análisis de pérdida de clientes.
Las tablas están ordenadas por fecha y hora de forma automática para facilitar la prioridad en el servicio.
