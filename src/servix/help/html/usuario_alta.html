Registro y Edición de Usuarios

Este formulario permite dar de alta a nuevos miembros en el sistema o modificar los datos de usuarios existentes.

Los campos marcados con un asterisco rojo (*) son obligatorios.

Instrucciones de Llenado

  • Datos Personales: El nombre y los apellidos no deben superar los 30 caracteres.
  • Teléfono: Debe contener exactamente 9 dígitos numéricos. El sistema resaltará el campo en rojo si no cumple esta condición.
  • Correo Electrónico: Se requiere un formato válido (ejemplo: usuario@dominio.com)
  • Contraseña: Debe introducir la misma contraseña en ambos campos de seguridad para evitar errores de escritura.
Seguridad: Las contraseñas se almacenan de forma encriptada. Ni siquiera el administrador puede ver su contenido original.

Atajos de Teclado

Para agilizar la introducción de datos, puede navegar por el formulario usando:

  • Enter: Salta al siguiente campo de texto.
  • ↑: Desplaza el cursor hacia arriba entre los campos.

Validación en Tiempo Real

Color de Fondo Significado
Blanco Dato correcto o campo vacío antes de validar.
Rojo claro Error de formato, exceso de caracteres (máx. 30) o campo vacío despues de validar.
